Graphics

1. What kind of photos are good to encode using JPEG?

JPEG is best for photos with many colors and smooth gradients, like landscapes and portraits.

2. What kind of photos are good to encode using GIF?

GIF is good for simple images with few colors, like logos, icons, and animations.

3. What is unique about the PNG format?

PNG keeps high quality with transparency, making it great for images needing a clear background.

4. How do you resize your photo to the size it should be displayed?

Resize your photo by using a photo editing app to set the right width and height before uploading.

5. Why should you resize photos using photo editing software instead of resizing it using CSS?

Use photo editing software because resizing with CSS only changes how it looks on screen but still loads the full large file, making pages slower.
