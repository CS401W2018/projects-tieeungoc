Colors

1. we can use tag names (p), IDs (#firstpar), classes (.blue), child selectors (footer p), and combinations (.blue.bigger). SPAN is for inline styling, and DIV groups block elements.

2. ID and Class: ID (#firstpar) is unique to one element, while class (.blue) can be used on multiple elements for shared styling.

3. SPAN and DIV: SPAN is inline, for styling text parts. DIV is block-level, for grouping elements.

4. DIV and HEADER, MAIN, FOOTER: DIV is a general block, while HEADER, MAIN, and FOOTER are semantic tags that define page structure. 5. Ways to Select a Color: Use named colors (blue), hex codes (#ff0000), RGB (rgb(255, 0, 0)), or HSL (hsl(120, 100%, 50%)).
