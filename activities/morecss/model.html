Box Model

1. Block tags (like div, p, h1) take up the whole line and push everything below. Inline tags (like span, a, strong) stay in the same line and only take up as much space as needed.

2. Margin: Space outside the element (pushing it away from others). Border: The outline around the element. Padding: Space inside the element (between content and border). CSS Units of Measurement

3. Fixed sizes: px, cm, in Flexible sizes: %, em, vw.

4. Use figure when adding images, charts, or videos with captions (figcaption), so they stay together and are easier to style.
