3. What are special entity characters?

Special entity characters are codes in HTML used to display characters that can't be typed directly into the source code.

Examples: &lt;, &gt;, &amp;, and &quot;

4. What are phrase tags?

  • Emphasis tag: highlights text
    Example: This is important text.
  • Marked tag: adds yellow background
    Example: This is highlighted text.
  • Strong tag: makes text bold for importance
    Example: This is important text.
  • Abbreviation tag: defines short forms
    Example: USA
  • Acronym tag: no longer used in HTML5
    Example: IBM
  • Definition tag: defines technical terms
    Example: HTML is used to make web pages.
  • Blockquote tag: long quote from another source
    "The only way to do great work is to love what you do." - Steve Jobs
  • Short quote tag: short quotation
    Example: He said, Hello!
  • Code tag: shows code in monospaced font
    Example: print("Hello")
  • Keyboard tag: shows keyboard input
    Example: Press Ctrl + C to copy.
  • Address tag: contact info block
