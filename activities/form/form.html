JavaScript Questions

1. What is JavaScript? How did it get its name?
JavaScript is a language that helps websites do things like animations, clicks, and popups. It was named "JavaScript" to get attention because Java was popular at the time.

2. What are the two ways you can run JavaScript?
You can run JavaScript in a web browser or on a server using a tool like Node.js.

3. What was the most popular front-end JS library?
jQuery was the most used JavaScript library before. It made coding easier.

4. What is the most popular front-end JS library today?
React is now the most used library for building websites with JavaScript.
