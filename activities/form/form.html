1. What is AJAX?

AJAX is a way to get or send data to a server without reloading the page. It helps make websites faster and smoother, like when a form submits and shows a message without refreshing.

2. Why put JavaScript in a separate file?

It helps keep things neat. HTML is easier to read, and we can reuse the same JavaScript on different pages. The browser can load it faster next time because it saves the file.
