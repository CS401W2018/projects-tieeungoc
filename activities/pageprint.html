Question:

1. Why do we need a viewport meta tag?

We use the viewport meta tag to make sure our website looks good on phones. If we don’t use it, the page might look too small or zoomed out.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

If we use the same tag and give it both “flex-direction: column;” and “flex-direction: row;”, the one written last will be used. So the second one has more power.

3. How do you change the CSS for a web page displayed on a phone?

To adjust the web page for phones, we use media queries. This helps change the layout for smaller screens.
