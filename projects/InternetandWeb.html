Internet

A global network that connects computers and allowing people to share information and communicate using standard rules.

Web

The web is a system of pages link together, as a collection of pages with resources and information that we can see, view and use on the internet with a browser.

How are the Internet and Web different?

The Internet is the global network that connects everything, and the Web is the part of it where you browse websites and pages.

ASCII and Unicode

ASCII is a small and only handles basic English characters and symbols. Unicode is larger and can handle characters from almost every language in the world, also emojis and special symbols.

What is UTF-8

UTF-8 is a way to turn text into numbers that computers can understand. It can handle all kinds of characters, including letters, symbols, and emojis, from any language. It's the most common system used for text on websites because it works with everything and is easy to use.

URL

A URL is a web address that tells our browser where to find a specific page, image, or video on the internet.

Domain Name

A domain name is the name of a website, like facebook.com that makes it easy to find online without using numbers.

HTTP

HTTP is the system of rules for sending and receiving information between your browser and a website, it is also a foundation of the World Wide Web and making the web work

HTTPS

HTTPS is a secure version of HTTP that keeps information private and protect them by encrypting the communication between our device and a website.

IP Address

IP address is a unique number that identifies a device on a network, such as a phone number for our computer or phone, so they can communicate with other devices to send and receive information

DNS

DNS is a system that changes website names like into IP address that computers use to find websites. It helps us visit websites using easy names instead remembering numbers.

W3C Recommendations

W3C Recommendations are rules made by the World Wide Web Consortium to help websites and tools work the same way on all browsers and devices. They make sure the web is easy to use for us and make sure it is reliable, and works for everyone.

XML, XHTML, HTML, HTML5

XML is a way to store and share data and used for organizing data, not for building the websites. XHTML is a version of HTML that is more strict and follows XML rules. It ensures cleaner and more organized code for web pages. HTML is a basic language used to create web pages. It adds structure to text, like headings, links, and paragraphs. HTML5 is the newest version of HTML. It adds features like videos, audio, and animations without extra software.
